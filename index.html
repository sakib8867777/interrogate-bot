How It Works

Integrate payments into your bot in five simple steps. The middleware handles the complex gateway routing so you don't have to.

1

Initiate Request

Your Telegram bot sends a standard JSON payment request to the SkyPay Middleware API containing user details and amount.

2

Gateway Creation

The Middleware securely authenticates and creates the payment intent with the core SkyPay Gateway.

3

User Checkout

You send the returned `payment_url` to the user. They open it and complete the transaction on the SkyPay checkout page.

4

Status Verification

The Gateway processes the payment and returns the final status to the Middleware for verification.

5

Webhook Delivery

The Middleware formats and fires a webhook back to your Telegram bot server to fulfill the user's order.

API Reference

Complete documentation for integrating the SkyPay Middleware into your bot application.

POST /api

Create a new payment request and generate a checkout URL for the user.

Request Body

  • webhook string
    The absolute URL where we will send the payment result.
  • name string
    Customer or username to display on checkout.
  • key string
    Your secret API brand key. Keep this highly secure.
  • amount string
    The transaction amount in default currency.
  • user_id string
    The unique Telegram ID of the user purchasing.
  • domain string
    Your primary gateway domain (e.g., https://pay.skypaybd.top).
JSON Request 
{
  "webhook": "https://your-bot.com/webhook",
  "name": "Customer Name",
  "key": "your-api-brand-secret-key",
  "amount": "100",
  "user_id": "123456789",
  "domain": "https://pay.skypaybd.top"
}
Response (200 OK) 
{
  "ok": true,
  "id": "9384758293",
  "payment_url": "https://middleware-domain.com/pay/9384758293",
  "status": "creating"
}

GET /pay/{id}

This endpoint handles the user checkout process. Send the user to the `payment_url` returned from the `/api` creation step. This route automatically redirects the user to the generated SkyPay checkout UI.

Telegram Bot Action 
1. Receive `payment_url` from POST /api
2. Send Inline Keyboard button to user:
   [ 💳 Pay Now ] -> URL: payment_url
3. User clicks and is redirected to checkout.

Webhook Processing

Once the transaction is resolved, the middleware will fire a POST request to your specified webhook URL. Your bot should listen on this URL to provide digital goods or update account balances.

Security Best Practices

When handling webhooks in production:

  • Always use HTTPS endpoints for your webhook listener.
  • Never expose your secret keys in client-side code.
  • Verify the user details internally before fulfilling the order.

Integration Examples

Quick start templates for popular backend languages.

PHP Implementation

Create a payment intent server-side using standard PHP cURL functions. This is ideal for traditional backend architectures.

PHP (cURL) 
<?php
$payload = json_encode([
    "webhook" => "https://your-bot.com/webhook",
    "name" => "Customer",
    "key" => "YOUR_SECRET_KEY",
    "amount" => "500",
    "user_id" => "123456789",
    "domain" => "https://pay.skypaybd.top"
]);

$ch = curl_init('https://middleware-domain.com/api');
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, $payload);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Content-Type: application/json'
]);

$response = curl_exec($ch);
curl_close($ch);

$result = json_decode($response, true);
if ($result['ok']) {
    echo "Send this URL to user: " . $result['payment_url'];
}
?>

Node.js Telegram Bot

A complete example using `node-telegram-bot-api` and the built-in native Javascript `fetch()` API to generate an inline keyboard payment button.

JavaScript (Node.js) 
const TelegramBot = require('node-telegram-bot-api');
const bot = new TelegramBot('YOUR_TELEGRAM_TOKEN', {polling: true});

bot.onText(/\/pay/, async (msg) => {
  const chatId = msg.chat.id;

  const response = await fetch('https://middleware-domain.com/api', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      webhook: "https://your-bot.com/webhook",
      name: msg.chat.first_name,
      key: "YOUR_SECRET_KEY",
      amount: "100",
      user_id: chatId.toString(),
      domain: "https://pay.skypaybd.top"
    })
  });

  const data = await response.json();

  if (data.ok) {
    bot.sendMessage(chatId, "Click below to complete your secure payment:", {
      reply_markup: {
        inline_keyboard: [[
          { text: "💳 Pay Now", url: data.payment_url }
        ]]
      }
    });
  }
});
